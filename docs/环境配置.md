## 获取并配置OpenAI API key

将`.env-back`文件复制为同目录下的 `.env`

随课堂会提供一个试用的 OPENAI API key. 

请将该key保存到`.env`文件中，

```
OPENAI_API_KEY="xxx"
```  

然后打开 `content/setup.ipynb` 文件。
